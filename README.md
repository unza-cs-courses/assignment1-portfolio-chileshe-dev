**Design Theme:**
A clean and modern responsive portfolio design with a minimal layout. The website uses a light color palette with a blue primary color and includes a dark/light mode toggle for better user experience. The layout focuses on readability and clear presentation of projects using card-based design.

### CSS Techniques Used

* [x] CSS Custom Properties
* [x] Flexbox
* [x] CSS Grid
* [x] Media Queries
* [x] Other: Responsive design, hover effects, dark/light mode toggle

### Challenges & Solutions

One challenge I encountered was implementing the dark/light mode toggle without breaking the design. Initially, the text in the projects section became invisible when switching to dark mode because the card background remained white while the text color changed. I solved this problem by replacing hardcoded colors with CSS variables and adjusting the project card background for dark mode. This ensured proper contrast and consistent styling across the site.

Another challenge was making the layout responsive. I solved this by using a mobile-first design approach with CSS Grid for the project cards and Flexbox for navigation and layout alignment. Media queries were added at 768px and 1024px breakpoints to adjust the layout for tablet and desktop screens.
