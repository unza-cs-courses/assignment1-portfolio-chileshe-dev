About Me

I am a Computer Science student with a strong interest in web development and software engineering. I enjoy designing and building responsive websites that provide meaningful solutions to real-world problems. My journey into programming started when I began learning HTML, CSS, and JavaScript. These technologies allowed me to create my first websites and sparked my interest in building digital products. Since then, I have expanded my skills into areas such as responsive web design, Git version control, and system analysis. Some of the projects I have worked on include a Bookstore Point of Sale system, a Housing Agent platform for managing real estate listings, and an Agent-Based Maze Solver project. These projects helped me develop problem-solving skills and understand how software systems are designed and implemented. My goal is to continue improving my development skills and eventually work as a professional software developer creating efficient, user-friendly, and scalable applications.
